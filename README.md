# nodejs-launcher
A simple nodejs launcher that executes a latest nodejs binary

## Confirmed work on:
* Glitch.com
* Replit.com

## Requirement
* x86_64 linux system/container
* [libcurl](https://curl.se/libcurl) installed with `curl` binary
